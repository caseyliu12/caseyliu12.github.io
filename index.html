Projects

Used Car Prediction

Capstone Project

Predicted the price of used cars in British market based on 9 features provided. For Mercedes C Class & Ford Focus cars which are two of the most popular cars in used car market, I also made classification models to predict what model of the used car is from.

The Linear Regression Model got 86% accuracy. The Ridge and Lasso didn't help improve the Linear Regression Model. The Random Forest Model preformed best with accuracy of 96%. All the classification models performed well. The Random Forest Model worked best with the score of 0.9906.

World Happiness Prediction

Group Project

Made different regressors to predict the happiness score based on 13 of the life features and did several multi-classifier models to predict the region. Used Tableau to create a visualization to show the features relate to each other and explored heat maps for each region to identify correlations between the different features and the happiness score. Got R2 score of 0.99 from linear regression model with Polynomial features and 0.92 from Random Forest Model.
